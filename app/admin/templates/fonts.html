說明

  • 台灣系統字型：macOS / Windows / Linux 預裝的繁體中文字型，如 PingFang TC、微軟正黑體、標楷體。
  • 開源 CJK 字型：Noto Sans/Serif TC、思源黑體、cwTeX、TW-Kai/Sung 等符合開放字型授權 (OFL) 的字型。建議裝 Noto TC 可以獲得最佳的繁中顯示效果。
  • PyMuPDF 內建：`china-t / china-ts / china-s / china-ss / helv / tiro / cour` — 最輕量、100% 相容，但字體美感較差，只在系統沒有更好選擇時使用。
  • 自訂字型：用上面的上傳表單加入 .ttf / .otf / .ttc；自訂字型會顯示在 PDF 編輯器字型下拉選單的最上方，並可從表格直接刪除。
