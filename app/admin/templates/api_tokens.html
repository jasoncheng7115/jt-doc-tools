呼叫本工具的 REST API（/api/*、/tools/*/submit 等）時要帶的 bearer token。 使用方式：Authorization: Bearer <token>。

強制檢查

關閉（預設）：寬限期，方便 web UI 與既有 script 無痛使用；不檢查 token。
開啟：沒帶有效 token 的 /api/* 呼叫一律拒絕 401；web UI 本身仍照常運作（不走 token）。

使用範例

# cURL
curl -X POST http://localhost:8765/api/llm-review \
  -H "Authorization: Bearer <YOUR_TOKEN>" \
  -F file=@your.pdf

# Python (httpx)
import httpx
r = httpx.post(
    "http://localhost:8765/api/convert-to-pdf",
    headers={"Authorization": "Bearer <YOUR_TOKEN>"},
    files={"file": open("doc.docx", "rb")},
)
