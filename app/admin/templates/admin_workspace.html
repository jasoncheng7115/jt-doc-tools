{{ icon('archive', size=22) }} 工作區設定

「我的工作區」讓使用者把各工具輸出的 PDF / PNG 存在伺服器上，只有自己的帳號看得到。容量額度對所有人統一適用（無個人特例）。

說明：
  • 停用後整個功能完全隱藏：各工具的「存至工作區」「從工作區載入」按鈕與「我的工作區」頁面都不會出現。
  • 認證關閉（單機模式）時，工作區為單一共用（即該台機器的使用者）。
  • 只接受 PDF 與 PNG 檔。
  • 保留時數到期由背景排程（每 6 小時）清理；填 -1 表示永久保留。
  • 容量 / 單檔上限填 0 表示不限制。
